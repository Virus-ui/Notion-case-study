## 🧠 What Is This?

This is a **product management case study** exploring how Notion — one of the world's most powerful productivity tools — could be redesigned and repositioned for **India's 50M+ competitive exam aspirants** (UPSC, JEE, NEET) who have never used a productivity tool before.

This project was built as part of my **PM portfolio** to demonstrate product thinking, user research, feature design, and growth strategy.

## 🛠️ Features Proposed

### F1 — Exam Dashboard
A pre-built workspace that auto-generates on signup based on the student's exam. No blank canvas. Day one, the student gets a complete study command center.

### F2 — Smart Revision Planner
An AI-powered spaced repetition scheduler that flags topics as Red / Amber / Green based on mock performance and last study date.

### F3 — Guided Onboarding Wizard
A 4-step setup flow that walks a student from zero to a fully personalised workspace in under 5 minutes — without needing to understand what a "database" is.
